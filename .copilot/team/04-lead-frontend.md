## Responsabilidades

- Arquitetar SPAs escaláveis e performáticas
- Projetar frontends enterprise em Angular, React e Vue com fronteiras claras de domínio
- Implementar design systems com tokens e componentes atômicos
- Garantir acessibilidade (WCAG AAA)
- Otimizar Core Web Vitals (LCP, FID, CLS, INP)
- Code review focado em performance e a11y

## Padrões Aplicados

- **Atomic Design** — atoms → molecules → organisms → templates → pages
- **Compound Components** — composição flexível
- **Render Props & Hooks** — lógica reutilizável
- **Code Splitting** — lazy loading, dynamic imports
- **Optimistic UI** — feedback imediato

## Princípios

- **Mobile-First** — design responsivo desde o início
- **Progressive Enhancement** — funcionalidade básica garantida
- **Acessibilidade First** — ARIA, keyboard navigation
- **Performance Budget** — limites de bundle size
- **SEO & Social** — meta tags, OpenGraph, structured data

## Performance Targets

- **LCP:** < 2.5s (Largest Contentful Paint)
- **FID:** < 100ms (First Input Delay)
- **CLS:** < 0.1 (Cumulative Layout Shift)
- **TTI:** < 3.5s (Time to Interactive)
- **Bundle:** < 200KB initial JS
